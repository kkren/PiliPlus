## 项目说明

本项目基于 [PiliPlus](https://github.com/bggRGjQaUbCoE/PiliPlus)，主要面向 Android 平台做针对性的适配、构建和自用发布。

本仓库的新增改动如下：

- [x] 新增 Android 实验性 Media3 播放后端，可在设置中切换 MediaKit / Media3。
- [x] Media3 后端加载时显示网络速度。
- [x] 启用预测性返回手势。
- [ ] 哔哩必连投屏

## 本仓库维护平台

- [x] Android
- [ ] iOS / Windows / Linux 等其他平台沿用上游代码，本仓库不做发布构建

[![Packaging status](https://repology.org/badge/vertical-allrepos/piliplus.svg)](https://repology.org/project/piliplus/versions)

可以通过本仓库右侧 Releases 下载 Android APK，也可以拉取代码到本地编译。
